## Repo Overview

- This repo is where I keep the dotfiles safe that I'm currently using for my MacBook.
- MacOS is my daily OS at the moment, because my back problems have gotten worse to the point where carrying my ASUS laptop around when I'm going to university – or whenever I want/need to work on the go – gives me terrible back pain.
- My OS is in the process of becoming pretty customised and I am using Neovim as my editor of choice.
  * I use several open source and non open source tools, including Kitty, Yabai, Sketchybar, etc.
  * I am far from being done with my current configurations yet, so quite a few changes will be made.

- A sneak peek of my setup will be added later

## Why are my dotfiles here?

- Because I like to customise and fuck around with my OS without having a lot of experience, I have accidentally erased my entire OS once already. This is why I've decided to create this repository, so that I can keep my setup safe.
- I already mentioned that I'm very new to this. Since I'm still learning, there is a high chance that I will mess up and that something will stop working or won't work to begin with.
  * By keeping my dotfiles here and tracking the changes made, I can safely mess around to my heart's content.
  * So this repo is also sort of like a safety net for me. If I do mess up one or several of my configurations, I can restore them more easily and save hours of painstaking work.

## Plans for this repository

- I honestly suck at staying consistent and commited (thanks, ADHD...), but I want to really try my best and track this repository as well as possible.
- My goal is to not only customise my MacOS as much as I can and set it up to my liking, but also to get to a point where other users can apply my dotfiles themselves.
